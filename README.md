# Swiggy-Excel-DashBoard
🍔 Swiggy Sales Analysis (Excel Project)

📌 Project Overview
This project is an end-to-end data analysis of Swiggy sales data using Microsoft Excel.
The goal is to extract meaningful insights such as top-performing cities, peak sales time, customer preferences, and trends.

🎯 Objectives
Analyze sales performance across different cities
Identify peak months, days, and time periods
Understand customer food preferences (Veg/Non-Veg)
Create an interactive dashboard for visualization
Generate business insights from raw data

📊 Tools Used
Microsoft Excel
Pivot Tables
Charts & Graphs
Formulas (VLOOKUP, IF, IFERROR, etc.)
Dashboard Design

📁 Dataset Details
The dataset contains:-
Order Date
City
Food Type (Veg/Non-Veg)
Sales Amount
Order Time / Day
📈 Key Analysis Performed
1. City-wise Sales Analysis
Identified top-performing cities
Compared revenue across locations
2. Monthly Sales Trend
Found highest sales month
Observed seasonal patterns
3. Day-wise Analysis
Identified peak day (e.g., weekend vs weekday)
4. Food Preference Analysis
Compared Veg vs Non-Veg sales
5. Dashboard Creation
Built an interactive dashboard using:
Pivot Charts
Slicers
KPI Cards
🔍 Key Insights
📍 Top Cities generate the highest revenue (e.g., Bangalore, Lucknow)
📅 Peak Month shows maximum order volume
📆 Weekend sales are higher than weekdays
🥗 Veg food category dominates overall orders
📊 Dashboard Features
Dynamic filtering using slicers
Easy-to-read KPI metrics
Visual charts for quick understanding
User-friendly design
🚀 How to Use
Download the Excel file from this repository
Open in Microsoft Excel
Use slicers to filter data
Explore insights from dashboard
